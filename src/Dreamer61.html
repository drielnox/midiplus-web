Dreamer61

Features:

  • 61 full-size standard semi-weighted piano keys
  • Support 64 notes to output simultaneously
  • Built-in 128 standard GM timbre
  • 6 self-modulated touching control buttons
  • Modulation & Pitch Bend touching control pads
  • 3X7 segment LED display screen
  • Providing a Data Select encoder which has mode select switch。
  • Various external control element:9V DC interface ，Power Select switch， standard USB port(used for provide power and communication)， Keyboard MIDI OUT interface， MIDI port， sustain pedal and expression pedal interface
  • Power supply：USB and power adapter 9v DC are available
  • Support Windows@ XP & Mac OS X

Technical Specifications:

  • Dimensions: 89×26×7(cm)
  • Weights: 8(kg)
